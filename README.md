## Install
npm install express --save
npm bootstrap --save

## terminal
server $ node app.js
client $ npm run serve

## Project
로컬 DB에 연결 되어있습니다. <br>
사용 하신다면, 본인의 DB 연결을 하시면 됩니다. <br>
It is connected to my local DB. If you use it, you can connect to your own DB.
